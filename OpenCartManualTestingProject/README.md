# OpenCart Manual Testing Project

This project contains comprehensive manual testing for the OpenCart e-commerce application.  
The test suite includes detailed test scenarios and test cases covering core e-commerce workflows and user account features.

## Objective
To validate the functionality, usability, and stability of the OpenCart application using systematic manual testing practices.

## Scope of Testing
- User Registration & Login
- Search & Product Display
- Add to Cart, Wishlist & Comparison
- Shopping Cart & Checkout
- Order Placement & Order History
- Returns, Transactions, Downloads, Reward Points
- Affiliate Module
- Newsletter & Contact Us
- Gift Certificates & Special Offers
- Header, Menu, Footer Navigation
- Multi-Currency Options

## Deliverables
- Test Scenarios & Test Cases
- Test Execution Results
- Bug Reporting

## Test Design
Each test case includes:
- Test Case ID  
- Module  
- Preconditions  
- Steps to Execute  
- Expected Result  
- Actual Result  
- Pass/Fail Status  
- Priority  
- Remarks / Defect Reference  

## Tools Used
- Excel (documentation)
- OpenCart Demo Website
- Chrome/Firefox Browser

## Outcome
A complete reusable manual regression suite for OpenCart, covering positive, negative, and boundary test conditions.
